# Hospital Management System

A simple Hospital Management System built using Node.js, Express.js and EJS.

## Features

- Register new patients
- Store patient details in a text file
- View all registered patients
- Responsive UI with modern styling

## Tech Stack

- Node.js
- Express.js
- EJS
- HTML
- CSS
- JS

## Installation

Clone the repository:

```bash
git clone https://github.com/ishitasinghal1611/MERN-Training-Hospital-management-project.git
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
node hospitalScript.js
```

Open in browser:

```
http://localhost:3000
```
